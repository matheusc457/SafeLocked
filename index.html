Why LockBox?

LockBox is a minimal and secure CLI tool designed to manage TOTP codes entirely offline. Your secrets never leave your machine.

Features

AES-256-GCM encryption
Argon2id key derivation
Background agent with memory-only master key
Clipboard auto-clear
Real-time TOTP watching
Fully offline operation

Security

Vaults are encrypted using AES-256-GCM and the encryption key is derived from your master password using Argon2id.

During unlocked sessions the key lives only in RAM through the background agent and is destroyed immediately when the vault is locked.
