Why LockBox?

LockBox is designed for users who want a simple, secure and fully offline TOTP manager. Your secrets stay on your machine — no cloud services, no tracking, no unnecessary complexity.

Features

AES-256-GCM encryption
Argon2id key derivation
Background agent with memory-only master key
Clipboard auto-clear
Real-time TOTP watching
Fully offline operation

Security

LockBox uses modern cryptographic primitives to protect your secrets. Vaults are encrypted using AES-256-GCM while the encryption key is derived from your master password using Argon2id.

The master key is stored only in RAM during active sessions via the background agent and destroyed immediately when the vault is locked.
